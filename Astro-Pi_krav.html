Esa's krav:

Our Program:

- relies on using the motion sensor and the camera
- contains no code that attempts to disrupt system functionality
- does not start a system process, nor runs another program
- does not use networking
- respects all the requirements concerning files and threads as well

I det at vi ikke kan tage videoerne af astronauterne ned til os, da bliver vores forsøåget meget mere upræcist.

Our Astro-Pi project is written in Python 3 and carries the name main.py. We do not rely on any libraries other than those listed in the phase 2 guide and the csv library. Our program monitors its running time and stops after three hours have elapsed. We do not make use of rudeness and bad language in our program.

In the security department, our program is fully understandable and there is no attempt to hide what a piece of code does.
